# Practice Exercises (Java)

This folder contains common coding exercises and algorithmic challenges for beginners to practice logical thinking and basic Java syntax.

## Exercise List

1. **Leap Year Checker**: [LeapYear.java](LeapYear.java) - Logic for determining leap years.
2. **Fibonacci Sequence**: [Fibonacci.java](Fibonacci.java) - Generating terms in the Fibonacci sequence.
3. **Prime Checker**: [PrimeCheck.java](PrimeCheck.java) - Testing if a number is prime.
4. **Palindrome Checker**: [Palindrome.java](Palindrome.java) - String manipulation to check for palindromes.
5. **Factorial**: [Factorial.java](Factorial.java) - Iterative calculation of n!.
6. **Right Triangle Pattern**: [RightTrianglePattern.java](RightTrianglePattern.java) - Basic logic for nested loops.
7. **Pyramid Pattern**: [PyramidPattern.java](PyramidPattern.java) - Advanced nested loops with alignment.
8. **Reverse String**: [ReverseString.java](ReverseString.java) - Manual implementation of string reversal.
9. **Find Max**: [FindMax.java](FindMax.java) - Iterating through an array to find the largest element.
10. **FizzBuzz**: [FizzBuzz.java](FizzBuzz.java) - Classic conditional logic challenge.
