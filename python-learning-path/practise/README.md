# Practice Exercises (Python)

This folder contains common coding exercises and algorithmic challenges for beginners to practice logical thinking and Pythonic syntax.

1. **Leap Year Checker**: [leap_year.py](leap_year.py) - Basic conditional logic for dates.
2. **Fibonacci Sequence**: [fibonacci.py](fibonacci.py) - Generating numbers in a sequence using loops.
3. **Prime Checker**: [prime_check.py](prime_check.py) - Prime number logic and optimization.
4. **Palindrome Checker**: [palindrome.py](palindrome.py) - Demonstrating Python slice notation `[::-1]`.
5. **Factorial**: [factorial.py](factorial.py) - Using recursion for math calculations.
6. **Right Triangle Pattern**: [right_triangle_pattern.py](right_triangle_pattern.py) - Loop-based string multiplication.

8. **Reverse String**: [reverse_string.py](reverse_string.py) - Exploring string immutability and reversal.
9. **Find Max**: [find_max.py](find_max.py) - Iterating through lists and using `max()`.
10. **FizzBuzz**: [fizz_buzz.py](fizz_buzz.py) - Standard modular arithmetic challenge.
