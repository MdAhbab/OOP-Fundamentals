# Basic Syntax in Python

Python is known for its readability and use of indentation to define code blocks.

## Key Concepts

### 1. Indentation
Unlike Java or C++, Python uses white space to define scope.
```python
if True:
    print("Inside block")
```

### 2. Comments
- `#` for single-line comments.
- `""" """` or `''' '''` for multi-line docstrings.

### 3. Naming Conventions (PEP 8)
- **Variables/Functions**: `snake_case`
- **Classes**: `PascalCase`
- **Constants**: `UPPER_SNAKE_CASE`
